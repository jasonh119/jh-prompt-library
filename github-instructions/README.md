# GitHub Instructions

This folder contains instructions and prompts for working with GitHub repositories, version control, and collaboration workflows.

## Files

- **git-basics.md** - Basic Git commands and workflows
- **git-branching.md** - Branch management strategies
- **git-collaboration.md** - Collaboration workflows (pull requests, code reviews)
- **git-troubleshooting.md** - Common Git issues and solutions
- **github-actions.md** - CI/CD with GitHub Actions
- **repository-setup.md** - Setting up new repositories

## Purpose

These instructions help with:
- Version control best practices
- GitHub workflow automation
- Repository management
- Collaboration guidelines
- Troubleshooting common Git issues
